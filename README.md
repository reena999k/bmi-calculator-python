# BMI Calculator (Python)

This project is a simple Body Mass Index (BMI) calculator built using Python.

## Features
1.Takes height in centimeters
2.Takes weight in kilograms
3.Calculates BMI
4.Shows health category (Underweight, Normal, Overweight, Obese)

## Technologies Used
- Python

## How to Run the Program

1. Install Python
2. Run the file

python bmi_calculator.py
